# 4계층에서 하는 일
- 송신자의 프로세스와 수신자의 프로세스를 연결하는 통신서비스 (프로그램과 프로그램)

<br>

# 4계층 프로토콜의 종류
### TCP 프로토콜
- 연결지향형 (더 복잡하다)

### UDP 프로토콜
- 비연결 지향형 (상대적으로 단순하다)

<br>

# 포트 번호의 특징
- 하나의 포트는 하나의 프로세스만 가능
- 일반적으로 정해져있지만 무조건 지켜야 하는 것은 아니다.

- HTTPS : 443번
- DNS : 53번 
- ... etc
